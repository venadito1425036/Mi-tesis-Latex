Tema: Modelos dinámicos tipo Lotka-Volterra con control óptimo y juegos de Stackelberg.


\subsection{Interpretación estratégica (juego de Stackelberg)}

El líder diseña el control \(u(t)\) anticipando que el sistema tumoral responderá de manera adaptativa.  
Por su parte, el seguidor elige \(v(t)\) con el objetivo de aumentar la resistencia minimizando sus costos internos.

Esto genera un conflicto natural:
\[
\text{terapia} \;\longleftrightarrow\; \text{adaptación tumoral}.
\]
